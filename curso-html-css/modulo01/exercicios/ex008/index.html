Principais formaçoes

Negrito / Destaque

Nessa parte vamnos falar sobre a tag <b> que intuito é deixar o texto em negrito. Para gerar destaque. (nao semântica).

Nesta parte, usaremos a tag <strong> para usar o texto em negrito. Criando assim um destaque. (forma semantica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i>(nao sem6antica).

Nesta frase temos um termo em ênfase usando a tag <em> (semantica).

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>.

Texto grande e pequeno

Estamos criando um texto grande um texto pequeno neste paragrafo. Usando as tags <big> e <small>.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas nao considerado. Usando a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois. Usando a tag <ins>.

Existe também o sublinhado com a tag <u>, mas ela nao é semantica.

Texto sobrescrito

Para inserir coisas do tipo: x22+3. Usamos a tag <sup>.

Texto subscrito

Para fazer coisas do tipo: H2O. Usando a tag <sub>.
